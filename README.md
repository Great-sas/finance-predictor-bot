#Finance Predictor Bot 

Telegram bot for currency forecasting and stock analysis using machine learning.

Features

- `/forecast` - USD/RUB exchange rate forecast for 7 days (ML model)
- `/stocks` - Current stock prices
- `/analyze SYMBOL` - Technical stock analysis with charts
- `/help` - Command help

Technologies

- Python 3.8+
- python-telegram-bot
- pandas, numpy, scikit-learn
- matplotlib, yfinance
- Machine Learning (Random Forest)
